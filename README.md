#Topico 1 - Preparacao do ambiente Linux

##Ambiente utilizado

VM local utilizando Linux Ubuntu

##Objetivo desta atividade

Preparar o ambiente Linux e organizar a primeira evidencia tecnica

##Estrutura Criada

Foi criada uma estrutura de diretorios para armazenar evidencias, comandos executados e documentacao tecnica

Estrutura:

    .topico-01/
        .evidencias/
        .comandos-topico-01.txt
        .README.md

    .produto-final/

##Comandos executados

pwd - mostra o diretorio
whoami - identifica o utilizador
hostname - mostra o nome da maquina
uname -a - apresenta informacoes do sistema Linux
ls - lista ficheiros e diretorios
date - mostra data e hora do sistema
mkdir - para criar diretorios
tree - para listar conteudo dos diretorios

##Diferenca entre VM, VPS e infraestrutura em nuvem

VM e uma maquina virtual executada localmente no computador
VPS e um servidor virtual hospedado remotamente
Infraestrutura em nuvem utiliza recursos distribuidos atraves da internet
